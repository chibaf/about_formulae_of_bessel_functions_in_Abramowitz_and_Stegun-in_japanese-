（以下敬称略）
Debyeの論文を探してたが、1909年のものは見つからなかった。代わりに1910年のはあった。1909年のは実数次数で次数を大きくしたときにハンケル関数がどう振る舞うかのもの。1910年のは次数が複素数の場合に拡張している。

ベッセルはハンケルの実部、ノイマンはハンケルの虚部の関係があります。（実変数の場合）
